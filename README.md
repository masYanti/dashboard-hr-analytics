# HR Analytics Dashboard

## Deskripsi
Dashboard analitik sumber daya manusia (HR Analytics) yang dibuat menggunakan Streamlit sebagai tugas mata kuliah Intelijen Bisnis.

## Fitur
- Dashboard KPI
- Filter berdasarkan Department, Gender, dan Attrition
- Analisis Attrition
- Analisis Department
- Analisis Gender
- Distribusi Usia
- Visualisasi interaktif menggunakan Plotly

## Dataset
Dataset yang digunakan adalah `cleaned_HR_Analytics.csv`.

## Cara Menjalankan

Install library:

```bash
pip install -r requirements.txt
```

Jalankan aplikasi:

```bash
streamlit run streamlit_app.py
```
